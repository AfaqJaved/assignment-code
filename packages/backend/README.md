## Description

Backend Application write in nestjs using graphql

## Project setup

```bash
$ pnpm install (make sure you are in the root directory of project (backend))
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# production mode
$ pnpm run start:prod
```

## Run linting and formatting

```bash
# formatting using prettier
$ pnpm run format

# linting using eslint
$ pnpm run lint

```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
